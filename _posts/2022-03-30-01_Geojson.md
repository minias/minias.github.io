# geojson 이란?

|파일 확장자|.json, .geojson|
|---|---|
|인터넷 미디어 타입|application/geo+json[1]|
|포맷 종류|GIS file format|
|표준|RFC 7946|
|오픈 포맷?|yes|
|웹사이트|geojson.org|

## 형태

```geojson
{
  "type": "FeatureCollection",
  "name": "cis3",
  "crs": {
    "type": "name",
    "properties": {
      "name": "urn:ogc:def:crs:OGC:1.3:CRS84"
    }
  },
  "features": [{
    "type": "Feature",
    "properties": {
    },
    "geometry": {
      "type": "Point",
      "coordinates": [
        127.011270708147,
        37.491137602209
      ]
    }
  }]
}
```
